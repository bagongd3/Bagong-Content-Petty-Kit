Auto internal tag linking and smart SEO pagination for WordPress publishers.

== Description ==

Bagong Content Petty Kit helps publishers:

* Auto internal linking based on post tags
* Smart SEO pagination (pages, words, H2)
* Optimized for long-form content
* Secure and performance friendly

== Installation ==

1. Upload plugin to /wp-content/plugins/
2. Activate via Plugins menu
3. Open Smart Toolkit menu
